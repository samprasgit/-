透明度设置    毛玻璃特效

- macOS 下的 iTerm2 基本快捷键与其他应用一致，以下是**一些特殊快捷键**。
   
   ```
   command + R/K，清空屏幕
   command + enter，进入/退出全屏
   command + D，垂直分屏
   command + shift + D，水平分屏
   command + U，变换透明度
   command + ; ，命令历史
   control + A，移动光标到行尾
   control + E，移动光标到行尾
   control + W，删除光标前的内容
   command + K，删除光标后的内容
   control + U，删除整行
   control + D，删除光标后的字符
   ```
